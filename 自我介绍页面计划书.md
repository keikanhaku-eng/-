# 自我介绍页面计划书

## 1. 页面目的

本页面的目的不是单纯罗列个人经历，而是让公司同事在短时间内了解：

- 我是谁
- 我目前具备哪些基础能力
- 我为什么选择 IT 行业
- 我在团队中倾向于怎样合作
- 我今后想朝什么方向成长

整体印象应控制在「诚实、稳重、前向き、愿意学习、容易一起工作」。

## 2. 推荐主题

建议主题：

> 从中国来到日本，在跨文化环境中学习、适应，并作为 IT 工程师继续成长。

这个主题适合你的原因：

- 你有中国留学生背景，来日本后完成了专门学校情报处理学习。
- 你已经取得 IT パスポート、ビジネス能力検定 3級、基本情報技術者等资格。
- 你对 Java、Git、MySQL、AI 等技术有持续学习意识。
- 你有较强的跨语言、跨文化沟通能力，也有主动组织和连接人的经验。
- 你的人生经历里有韧性，但不需要在公司页面中直接暴露私人创伤。

## 3. 页面结构建议

### 3.1 开头：简短问候

目标：让读者感到自然、亲切，不要一开始就写得太重。

可表达内容：

- 姓名：敬涵博 / ケイ
- 出身：中国
- 现在：在日本 IT 企业作为社会人开始工作
- 一句话定位：正在学习并成长中的新人 IT 工程师

日语表达方向：

> はじめまして。中国出身の敬涵博（ケイ）です。日本で情報処理を学び、現在はITエンジニアとして成長していきたいと考えています。

### 3.2 背景：来日与学习经历

目标：说明你不是随便进入 IT，而是有持续适应和学习的过程。

可表达内容：

- 来到日本后进入专门学校学习情報処理。
- 在跨文化环境中适应学习、生活和沟通。
- 通过学习逐渐确认自己想在 IT 行业发展。

不要写：

- 过度强调痛苦经历
- 具体人际关系冲突
- 对学校或他人的负面评价

### 3.3 技术与资格

目标：给出可信的职业基础。

可表达内容：

- IT パスポート
- ビジネス能力検定 3級
- 基本情報技術者
- Java / Spring Boot 基础学习中
- Git 基础
- MySQL / SQL 学习中
- 对 AI 技术和开发效率提升有兴趣

表达时避免夸张，建议用「現在学習中」「基礎を身につけています」「今後さらに深めたい」。

### 3.4 自己的强项

目标：把个人特质转化为公司能理解的工作价值。

推荐写三点：

1. 持续学习能力
2. 跨文化沟通能力
3. 责任感与主动性

可用内容：

- 能用日语在日本环境中学习和工作。
- 能和不同年龄、文化背景的人建立联系。
- 遇到问题时倾向于先理解情况、承担责任，再寻找解决方法。
- 对别人交给自己的事比较认真。

不建议写：

- “我很会看人”
- “我社交能力很强”
- “我经历过很多背叛”
- “我比同龄人更成熟”

这些内容即使有真实成分，在公司页面里也容易显得过重或自我评价过高。

### 3.5 兴趣爱好

目标：让同事觉得你是一个具体的人，而不是只有技术标签。

适合写的兴趣：

- 做饭，特别是北京烤鸭
- Jazz
- 电影
- 游戏
- 健身
- 阅读技术资料或 AI 相关内容

推荐重点写「做饭」。

原因：这比抽烟斗、复杂人际经历、情绪故事更安全，也更容易让人记住。比如「从零做北京烤鸭」是一个很有画面感的点。

### 3.6 今后的目标

目标：让页面收在职业成长上。

可表达内容：

- 先扎实掌握 Java、SQL、业务知识。
- 尽快理解公司开发流程和团队规则。
- 成为能稳定完成任务、也能主动改善问题的工程师。
- 将来希望结合 AI 工具提升开发效率，但基础能力也会继续打牢。

推荐日语方向：

> まずはJavaやSQL、業務知識を着実に身につけ、チームに貢献できるエンジニアになることを目標にしています。将来的には、AIなどの新しい技術も活用しながら、より効率よく価値を生み出せる人材を目指したいです。

## 4. 页面风格建议

### 推荐风格

- 清爽
- 稳重
- 轻微个人感
- 不要太花
- 不要过度自夸
- 不要写成履历书

### 推荐版面

1. 顶部：姓名 + 一句话介绍
2. Profile：基本信息
3. Background：来日与学习
4. Skills：资格与技术
5. Strengths：强项
6. Hobby：兴趣
7. Goal：今后的目标

## 5. 不建议公开写入的内容

以下内容可以作为你理解自己的材料，但不适合放进公司自我介绍页面：

- 抑郁经历的具体经过
- 恋爱或暧昧关系
- 对特定人物的评价
- 家庭创伤细节
- 对过去学校、同学、朋友的负面描写
- 强烈情绪化表达
- “我经历过很多痛苦，所以……”这类卖惨式表达

如果需要表达这些经历带来的成长，可以转化为：

- 来日后经历过不少适应上的挑战，因此更重视稳定、责任和持续学习。
- 在不同文化和环境中生活后，更能理解沟通方式和价值观差异。
- 过去的经历让我更珍惜团队中的信任、尊重和边界感。

## 6. 推荐页面核心文案方向

中文草稿：

> 我是敬涵博，来自中国。来到日本后，我在专门学校学习情报处理，并取得了 IT パスポート、ビジネス能力検定 3級、基本情報技術者等资格。现在作为新人 IT 工程师，我正在继续学习 Java、SQL 和开发流程，希望尽快理解业务并为团队做出稳定贡献。
>
> 我认为自己的特点是适应力、持续学习能力和跨文化沟通能力。来到日本后，我在学习、生活和人际交流中不断适应新的环境，也逐渐学会了更主动、更负责地面对问题。
>
> 工作之外，我喜欢做饭、听 Jazz、看电影，也对 AI 技术和开发效率提升很感兴趣。今后我希望在打好基础的同时，逐步成长为既能理解技术，也能理解团队和业务的工程师。

日语草稿：

> はじめまして。中国出身の敬涵博（ケイ）です。日本で情報処理を学び、ITパスポート、ビジネス能力検定3級、基本情報技術者などの資格を取得しました。現在は新人ITエンジニアとして、JavaやSQL、開発の流れを着実に学びながら、早くチームに貢献できるよう努力しています。
>
> 私の強みは、環境への適応力、継続して学ぶ姿勢、そして異なる文化や価値観を持つ人とコミュニケーションを取る力です。日本での生活や学習を通して、新しい環境の中でも自分から動き、責任を持って物事に取り組む大切さを学びました。
>
> 仕事以外では、料理、Jazz、映画などが好きです。特に料理は、人と交流するきっかけにもなる大切な趣味です。今後は基礎を大切にしながら、AIなどの新しい技術にも関心を持ち、技術とチームの両方に貢献できるエンジニアを目指していきたいです。

## 7. 结论

这个自我介绍页面最应该表达的是：

> 「中国から日本に来て、学び続けながらITエンジニアとして成長していく、誠実で前向きな新人」

不需要把人生全部讲出来。公司页面里，最有力的表达不是暴露伤口，而是让别人看到你已经把经历沉淀成了学习力、责任感和适应力。
